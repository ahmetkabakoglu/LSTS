# LSTS

LSTS, laptop servis süreçlerini dijital ortamda yönetmek için geliştirilmiş rol tabanlı bir servis takip sistemidir.

## Kısa Özet

Uygulama; müşteri, cihaz, ticket ve parça süreçlerini tek panel üzerinden yönetmeyi amaçlar. Teknik servis akışını daha düzenli, izlenebilir ve yönetilebilir hale getirir.

## Temel Özellikler

- JWT tabanlı kimlik doğrulama
- Rol bazlı yetkilendirme
- Müşteri ve cihaz kaydı
- Ticket oluşturma ve durum takibi
- Parça ve parça talep yönetimi
- Public ticket sorgulama
- Oracle Database uyumlu backend
- Oracle Cloud üzerinde yayınlanabilir yapı

## Teknolojiler

- Vue 3
- TypeScript
- Vuetify
- .NET 10 Minimal API
- Oracle Database
- Nginx
- systemd

## Proje Yapısı

```text
LSTS/
├─ lsts-ui/

   └─ Lsts.Api/
```

## Amaç

Servis kabul, onarım takibi, parça yönetimi ve müşteri sorgulama süreçlerini tek bir sistem altında toplamak.
